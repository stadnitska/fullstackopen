## 0.6 New note in SPA

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User writes note and clicks Save

    browser->>browser: JavaScript handles submit event
    browser->>server: POST /exampleapp/new_note_spa
    activate server
    Note right of server: Server saves the new note
    server-->>browser: 201 Created
    deactivate server

    Note right of browser: Browser updates UI without page reload
```
